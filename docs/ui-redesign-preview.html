Design rationale

What changed vs. today

  • NavigationSplitView replaces the macOS preset sidebar + grid layout
  • Sidebar shows live tally state as a colored dot next to each camera — quick scan
  • System blue (#0A84FF) accent instead of pink — feels native, matches iOS
  • True black background — easier on the eyes in a dark booth, OLED-friendly
  • Tap a tile to open a full-screen detail with large +/− controls (sheet on iPhone)
  • Card-flip "settings back" is gone — settings live in an inspector / detail sheet instead

iPad-specific

  • Sidebar: cameras, presets, settings. Translucent material that blurs the wall behind
  • Toolbar large title ("Cameras") + status pills (TSL state, recording count)
  • 3-column grid at 12.9" landscape; collapses to 2 on iPad mini / portrait
  • Tally state is the loudest visual: 2px colored stroke + soft glow, never subtle
  • Hover (Magic Keyboard / trackpad) reveals quick actions on each tile

iPhone-specific

  • Tab bar: Cameras / Presets / Tally / Settings — one-thumb reach
  • Cards are full-width stacked rows, metrics at smaller scale
  • Tally chip moves to the right of the camera name — visible in the list
  • Tap a row → push to a single-camera control screen with large finger-friendly +/−
  • Pull-to-refresh re-runs Bonjour discovery

Apple HIG choices

  • SF Pro Display for titles, SF Pro Text for body, SF Mono for tabular numerics
  • System materials (thick + thin) for sidebar and toolbar
  • Standard corner radii: 22 for cards, 14 for buttons, 10 for chips
  • Touch targets ≥ 44pt where interactive (REC button = 44pt tall)
  • Semantic colors (systemRed, systemGreen) so the UI follows accessibility settings

What's preserved

  • All six metrics: Av · ISO · Tv · AE · WB · ND
  • Tally PGM/PVW indicators with colored glow
  • Per-camera REC/STOP, identify, remove, TSL index assignment
  • OLED preview, camera positions assignment
  • Preset system: list, recall, edit-with-checkmarks
  • Firmware update entry (settings tab on iPhone, toolbar on iPad)

Open questions

  • Does the iPad replace the director Mac, or run alongside it?
  • Does the iPad need wired Ethernet (USB-C dock) for tally reliability?
  • Should the iPhone be a true controller, or only a "glance" app?
  • Single-camera mode (full-screen one tile) needed for FOH operators?
  • Live thumbnail / video preview embed — out of scope or v2?
